# E2E
Runs Backend with ./
### `cd e2e-backend`
### `npm install --save`
### `node index.js`
Backend is running now you will see in your terminal
### `Server is running on port: 8080`
### `Successfully connected to MongoDB`.


Run Frontend With ./
### `cd e2e-frontend`
### `npm install --save`
### `npm start`

### `frontend-app is runing on Port 8080`
